# Active Web & Full-Stack Skills Guide

This guide details the 22 active skills enabled in your Antigravity IDE environment. You can trigger any of these skills by using `@skill-name` in your chat prompts.

---

## 🌐 The "Web Wizard" Pack

| Skill | What it's about | Example Prompt |
| :--- | :--- | :--- |
| `@frontend-design` | Base layer for visual aesthetics, layout conventions, and UI design standards. | `Use @frontend-design to review the spacing and typography hierarchy of the navbar component.` |
| `@react-best-practices` | Vercel-aligned React & Next.js performance and code structure rules. | `Use @react-best-practices to optimize rendering performance in this React list component.` |
| `@react-patterns` | Modern React architectural patterns, custom hooks, and state management. | `Use @react-patterns to refactor this class component into a functional component using hooks.` |
| `@nextjs-best-practices` | App Router patterns, Server Components (RSC), metadata, and caching conventions. | `Use @nextjs-best-practices to set up dynamic SEO metadata inside app/blog/[id]/page.tsx.` |
| `@tailwind-patterns` | Tailwind CSS v4 styling rules, custom configuration, utility design tokens. | `Use @tailwind-patterns to apply a responsive grid layout to the product collection page.` |
| `@form-cro` | Optimizing forms for user conversion, validation states, and input UX. | `Use @form-cro to audit the checkout form for layout accessibility and error visibility.` |
| `@seo-audit` | Best practices for search engine optimization, semantic HTML, and metadata. | `Use @seo-audit to check if stitch-assets/homepage_desktop.html meets modern SEO standards.` |
